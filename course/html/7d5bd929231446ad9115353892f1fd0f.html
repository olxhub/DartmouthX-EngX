Lesson: Determinancy 

What is determinancy? And how does it apply to trusses? Determinancy has to do with the number of supports and members, in this case in a truss. How many supports and members do you need to create a stable truss? Is it possible to have too many supports and members? These are a few of the questions we'll explore.
